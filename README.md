# Automação com PyAutoGUI

Projeto em Python que realiza **automação de cadastro de produtos em um sistema web**, utilizando controle automático do mouse e teclado.



### Sobre

O **automacao-pyautogui** é um projeto desenvolvido para demonstrar como automatizar tarefas repetitivas em sistemas web utilizando **Python**.

A automação realiza as seguintes etapas:

* Abre o navegador automaticamente
* Acessa o sistema da empresa
* Realiza o login
* Lê uma base de dados contendo produtos
* Preenche automaticamente os campos do sistema
* Cadastra todos os produtos da planilha de forma automática

Esse tipo de automação é muito utilizado para **ganho de produtividade**, evitando trabalho manual repetitivo.

### Bibliotecas utilizadas

* **PyAutoGUI** — automação de mouse e teclado
* **Pandas** — leitura e manipulação de dados da planilha
* **Time** — controle de pausas para sincronizar a automação



## Como Executar

### 1° Clone o repositório

```
git clone https://github.com/seu-usuario/automacao-pyautogui
```

### 2° Entre na pasta do projeto

```
cd automacao-pyautogui
```

### 3° Instale as bibliotecas necessárias

```
pip install pyautogui
pip install pandas
```

### 4° Execute o script

```
python automacao.py
```
